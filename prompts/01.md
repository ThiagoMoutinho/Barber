Você está trabalhando em um projeto de SaaS de agendamento 
para barbearias, onde o usuário pode se cadastrar, fazer login,
agendar um horário com uma barbearia, visualizar seus agendamentos
e cancelar um agendamento.

- Next.js
- Tailwind CSS

- Prisma 7 (veja o arquivo @prisma\schema.prisma)

- Docker
- shadcn-ui

## Requisitos funcionais
- O botão de reservar não deve fazer nada.
- O botão de "Copiar" telefone deve copiar para o clipboard o número de telefone da barbearia.
- Os dados da barbearia devem ser recuperados do banco de dados, usando o ID recebido na URL.
- A imagem do banner da página no topo deve ser a imagem da barbearia no banco de dados (imageURL).
- O botão de "Voltar" do banner da página deve voltar á tela anterior.
- Para cada serviço da barbearia recuperado do banco de de dados, renderize o @components\ui\servece-item.tsx (Crie o componente). Receba o serviço como prop.
- O usuário pode se cadastrar fornecendo nome, email e senha.
- O usuário pode fazer login fornecendo email e senha.
- O usuário pode agendar um horário com uma barbearia, fornecendo data, hora e barbearia.
- O usuário pode visualizar seus agendamentos.
- O usuário pode cancelar um agendamento.
