## Requisitos Funcionais

- Deve ser exibido o botão de login se o usuário não estiver autenticado.
- Deve ser exibido o botão de logout se o usuário estiver autenticado.
- Deve ser exibido o botão de perfil se o usuário estiver autenticado.
- Deve ser exibido o botão de agendar se o usuário estiver autenticado.
- Deve ser exibido o botão de ver agendamentos se o usuário estiver autenticado.
- Deve ser exibido o botão de cancelar agendamento se o usuário estiver autenticado.
- Se ele estiver logado, exiba o Avatar, nome e email (por enquanto, tudo had-coded)
- Botão de "agendamentos" deve levar para "/bookings".
- Botão de categorias (cabelo, barba, etc.) deve levar para "/barbershops?search={termo}". Veja o exemplo em @components\quick-search.tsx
- use o componente Sheet do shadcn para exibir o menu.
- Botão de "Sair" da conta não deve fazer nada, por enquanto.
- Esse componente deve ser aberto ao clicar no botão de menu do @components\header.tsx
