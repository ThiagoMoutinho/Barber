## Tarefas

- Criar um sheet de cancelamento de reserva que é exibido quando o usuário clica no @components\booking-item.tsx @
- A interface deve ser exatamente oque está no figma em https://www.figma.com/design/sFeJ4USRBec6gwfDxf7ZaL/Aparatus-%7C-Alunos--c%C3%B3pia-?node-id=78-2076&t=Uog2zpTv0d9OKeg4-4

## Requisitos funcionais

- Ao clicar em "Cancelar Reserva" no @components\booking-item.tsx @ exiba um alert Dialog solicitando a confirmação do cancelamento 
- Se o usuário confirmar o cancelamento, a reserva deve ser cancelada e um toast deve ser exibido confirmando o cancelamento
- Se o usuário cancelar a operação, nada deve acontecer
- Exiba os dados da barbearia e do serviço no sheet. Recupere esses dados do banco de dados.
-  O usuário deve poder ver os dados da barbearia e do serviço antes de confirmar o cancelamento.
- Os botões de "Copiar" telefone devem copiar os telefones para o clipboar.
- A imagem do mapa é estática @public\map.png.
- O agendamento é considerado "Confirmado" quando a data é no futuro, e "Finalizado" quando está no passado ou quando ele está cancelado. Crie uma função utilitária que retorne essa informação, e use ela também em @components\booking-item.tsx.
- Use o componente @app\barbershops\[id]\_components\phone-item.tsxn.
- O botão de "voltar" deve fechar o sheet.

## Requisitos técnicos.

- Crie uma serve action que faça o cancelamento da reserva.
- Só pode cancelar uma hora antes do agendamento.
- Caso tente cancelar depois da hora, exiba um toast informando que não é possível cancelar.
- Se o cancelamento for bem-sucedido, exiba um toast confirmando o cancelamento.
- Crie um componente chamado "booking-summary" que renderize https://www.figma.com/design/sFeJ4USRBec6gwfDxf7ZaL/Aparatus-%7C-Alunos--c%C3%B3pia-?node-id=235-348&t=b97tqmeQRHpDjZZ8-4. Use ele também em @components\ui\service-item.tsx
